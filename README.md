Problem Statement

The COVID-19 pandemic generated massive real-time data across countries, including cases, deaths, and recoveries.
Manual analysis of such large datasets is time-consuming and inefficient.
This project aims to automate analysis and provide meaningful insights through visualization.

2. Dataset Description

The dataset is taken from Kaggle and contains real-world COVID-19 data published by trusted organizations.
It includes key attributes such as date, country, confirmed cases, deaths, and recoveries.
Only relevant columns are selected to simplify analysis and improve performance.

3. Objective

The main objective is to perform Exploratory Data Analysis (EDA) and build an interactive dashboard.
The system helps in identifying trends, patterns, and relationships in COVID-19 data.
It also supports better understanding through visual insights and comparisons.

4. Data Collection

The dataset is loaded from a Kaggle source in CSV format.
Only required columns are selected for focused analysis.
This step ensures efficient data handling and reduces unnecessary complexity.

5. Data Cleaning & Preprocessing

Data is cleaned by handling missing values and removing duplicates.
The date column is converted into proper datetime format for analysis.
Numerical columns are validated to ensure accurate calculations.

6. Descriptive Statistics

Basic statistics like total confirmed cases, deaths, and recoveries are calculated.
This provides an overview of the dataset and its distribution.
It helps in understanding the overall impact of COVID-19 globally.

7. COVID Trend Analysis

The project analyzes how confirmed cases change over time.
It helps identify peaks and waves during different periods.
This reveals growth patterns and spread behavior of the virus.

8. Country-Based Analysis

The system identifies the most affected countries based on case numbers.
It compares countries using confirmed cases, deaths, and recoveries.
This helps in understanding regional impact and severity.

9. Dashboard-Based Analysis

An interactive dashboard is built using Plotly Dash for visualization.
Users can select country and date range for dynamic analysis.
It makes data exploration simple, interactive, and user-friendly.

10. Relationship Analysis

Relationships between variables like confirmed vs deaths are analyzed.
Scatter plots help visualize correlations between different factors.
This helps in understanding how variables influence each other.

11. Data Visualization

Multiple charts are used to represent data clearly and effectively.
Line charts show trends, bar charts compare countries, and pie charts show distribution.
Heatmaps and scatter plots provide deeper analytical insights.

12. Dashboard Insights

The dashboard helps identify high-risk countries and critical periods.
It shows recovery trends and overall pandemic progression.
These insights help in better decision-making and understanding.

13. Insights & Conclusion

The project highlights key trends and patterns in COVID-19 data.
It demonstrates the importance of data analysis in global health crises.
Visual dashboards make complex data easy to understand and interpret.
